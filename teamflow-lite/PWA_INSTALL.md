# TeamFlow App 安装与使用

TeamFlow 现在是可安装的 PWA。它既可以通过网址访问，也可以安装到手机或电脑桌面，以独立 App 窗口运行。

## Android 安装

1. 使用 Chrome 或 Edge 打开线上 TeamFlow 地址。
2. 点击页面中的“安装 App”，或打开浏览器菜单。
3. 选择“安装应用”或“添加到主屏幕”。
4. 确认后，TeamFlow 图标会出现在手机桌面。

## iPhone / iPad 安装

1. 使用 Safari 打开线上 TeamFlow 地址。
2. 点击底部“分享”按钮。
3. 向下找到“添加到主屏幕”。
4. 点击右上角“添加”。

安装后请从桌面图标启动 TeamFlow。进入左侧“App 与通知”可以查看安装状态和开启系统通知。

## Windows / macOS 安装

使用 Chrome 或 Edge 打开 TeamFlow，点击地址栏右侧的安装图标，或者点击产品中的“安装 App”。安装后会出现在开始菜单、应用列表和桌面快捷方式中。

## 与应用商店 App 的区别

| 项目 | TeamFlow PWA | 原生 App |
| --- | --- | --- |
| 安装入口 | 网址或浏览器菜单 | App Store / 应用商店 |
| 更新方式 | 打开后自动更新 | 商店下载更新 |
| 手机和电脑 | 一套产品同时支持 | 通常需要分别开发 |
| 桌面图标和独立窗口 | 支持 | 支持 |
| 离线启动 | 支持基础界面；业务数据需联网 | 取决于原生实现 |
| 系统通知 | 支持，但受浏览器和系统权限限制 | 支持更完整 |
| 蓝牙、通讯录等深度设备能力 | 能力有限 | 能力更完整 |
| 应用商店审核 | 不需要 | 需要 |

## TeamFlow 提醒说明

- 服务器会持续扫描到期和逾期事项，并通过飞书或通用 Webhook 主动推送。
- 安装后的 App 在打开时，可以使用手机系统通知提示当前待跟进事项。
- 即使 App 没有打开，飞书/Webhook 后台提醒仍然有效。
- 完全独立的手机后台 Push 需要另外配置 Web Push 订阅服务，目前 Service Worker 已预留 Push 接收入口。
